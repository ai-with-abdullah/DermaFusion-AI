# Code of Conduct

## Our Pledge

We are committed to making participation in the DermaFusion-AI project a respectful and harassment-free experience for everyone, regardless of background, level of experience, nationality, or identity.

## Our Standards

**Positive behaviour includes:**
- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the research community
- Showing empathy towards other contributors

**Unacceptable behaviour includes:**
- Harassment, insults, or derogatory comments
- Publishing others' private information without permission
- Any conduct that would be considered inappropriate in a professional setting

## Responsibilities

Project maintainers are responsible for clarifying standards of acceptable behaviour and may remove, edit, or reject contributions that do not align with this Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces — GitHub Issues, Pull Requests, Discussions, and any public representation of this project.

## Enforcement

Instances of abusive or harassing behaviour may be reported by contacting the project maintainer:

**Muhammad Abdullah** — via GitHub Issues or the contact listed in the repository.

Maintainers will review and investigate all complaints and respond in a way that is deemed necessary and appropriate.
